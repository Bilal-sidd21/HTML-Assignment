# University Admission Form

## Project Overview

This project is a **University Admission Form** developed using **HTML5**. It is designed to collect student admission information in a structured and user-friendly format without using CSS or JavaScript.

## Features

* University logo and heading
* Personal Information section
* Contact Information section
* Academic Information section
* Document Upload section
* Declaration section
* Form validation using HTML validation attributes

## HTML Elements Used

* `<form>`
* `<fieldset>`
* `<legend>`
* `<label>`
* `<input>`
* `<select>`
* `<option>`
* `<datalist>`
* `<textarea>`
* `<button>` / `<input type="submit">`

## Input Types Used

* Text
* Email
* Date
* Tel
* Number
* Radio
* Checkbox
* File
* Submit

## Validation Attributes Used

* `required`
* `minlength`
* `maxlength`
* `min`
* `max`
* `pattern`
* `accept`

## Project Structure

* University Header (Logo, Name, Address)
* Personal Information
* Contact Information
* Academic Information
* Document Upload
* Declaration
* Submit Button

## Purpose

The purpose of this project is to demonstrate the use of HTML forms and form elements for creating a complete university admission application. It also shows how HTML validation can be used to ensure users provide valid information before submitting the form.
